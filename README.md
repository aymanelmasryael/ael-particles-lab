**High-density particle simulation engine** with 100 taxonomy classes across 10 physics layers, real-time rendering, and AEL Digital Framework integration.

## Features

- **100 Particle Classes**: 10 layers × 10 particles each (Physics, Visual, Procedural, AI Behavioral, Fluid Dynamics, GPU Compute, State Lifecycle, Environment Forces, Hybrid Composite, Advanced Quantum)
- **Hybrid Engine v3.0**: High-density batched rendering (2,000–10,000 particles per layer)
- **Real-time Layer Switching**: Click any particle class to instantly switch physics behaviors
- **Spatial Grid Optimization**: Fast particle updates with boundary clamping
- **Connection Lines**: Visual spring constraints on Hybrid Composite layer
- **Mouse Interaction**: Particles respond to cursor in AI Behavioral layer
- **Sovereign Identity Block**: AEL Digital Framework with owner verification

## Tech Stack

- **HTML5** — Semantic structure
- **Tailwind CSS** — Utility-first styling
- **CSS3** — Glassmorphism, cyber corners, scanline overlay
- **JavaScript** — Hybrid rendering engine, particle taxonomy system
- **Deployment**: GitHub Pages

## Usage

1. Click any particle class in the left taxonomy panel
2. Watch the rendering engine switch physics behaviors instantly
3. Hover over the canvas (AI Behavioral layer) to interact with particles
4. View telemetry data in the HUD overlay
5. Verify the Sovereign Identity Block on the right panel
